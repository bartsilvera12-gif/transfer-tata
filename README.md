# Transfer Tatá

Landing page para Transfer Tatá — servicio de traslados premium.

## Contenido

- `Transfer Tata Landing.html` — diseño base de la landing (HTML/CSS/JS standalone) entregado por diseño.
- `uploads/LOGO Transfer TATA.pdf` — logo oficial.

## Stack

Por ahora el proyecto vive como un HTML estático autocontenido (fuentes desde Google Fonts, sin dependencias). Próximos pasos posibles:

- Migrar a Next.js + Tailwind para componentizar la landing.
- Agregar formulario de reservas conectado a backend.
- Deploy a Vercel.

## Desarrollo local

Abrir `Transfer Tata Landing.html` directamente en el navegador, o servir con cualquier servidor estático:

```bash
npx serve .
```
